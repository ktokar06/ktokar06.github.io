Tournament Management System

О проекте

Полнофункциональное веб-приложение для управления турнирами по беспилотным летательным аппаратам (БПЛА). Система для организации, проведения и архивации результатов соревнований.

Задача

Разработать веб-приложение для автоматизации процесса организации и проведения турниров по БПЛА. Система должна упростить регистрацию участников, ведение турнирных таблиц, подсчет результатов и архивацию данных.

Результаты

1. Полнофункциональная система управления турнирами с поддержкой двух этапов: отборочного и финального

2. Реализован полный CRUD функционал для управления турниров

3. Разработан пользовательский интерфейс с навигацией между разделами системы

4. Интеграция с базой данных для хранения и извлечения турнирных данных

5. Система успешно используется для проведения турниров внутри колледжа

Чему научился

Научился проектировать и разрабатывать полный цикл веб-приложения от планирования до реализации. Приобрел опыт работы с Java и Spring Boot, проектированием базы данных и разработкой пользовательского интерфейса.
