О проекте

Проект по тестированию REST API. Проведение CRUD операций, создание тест-кейсов, чек-листов, баг-репортов, а также создание окружений для тестирования.

Задача

Обеспечение качества работы REST API, проверка корректности обработки данных.

Результаты

1. Создана система тестирования REST API с покрытием основных сценариев

2. Разработаны тест-кейсы для проверки валидации данных, обработки ошибок и производительности API

3. Проведено полное тестирование API эндпоинтов с использованием Postman

Чему научился

Научился тестировать REST API, проверять валидацию данных. Приобрел опыт создания тест-кейсов и чек-листов для API тестирования.

Технологии

Postman, REST API, JSON, Test-Case
